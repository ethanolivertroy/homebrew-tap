# Homebrew Tap

Personal Homebrew tap for my CLI tools.

## Installation

```bash
brew tap ethanolivertroy/tap
```

## Formulas

### cmvp

Terminal UI for searching NIST CMVP validated cryptographic modules.

```bash
brew install ethanolivertroy/tap/cmvp
```

Or directly:

```bash
brew install ethanolivertroy/tap/cmvp
```
